=========================
RELEASE NOTES
=========================
This release is intended as a bridge to work with DNN7. DNN7 does not yet exhibit any
breaking changes other than minor UI issues specifically dealing with the the Menu.

There are currently two flavors of install.
1. Rad Editor Supported - DNN introduced the Rad RichTextEditor in DNN6. The RadEditor installation provides that support.
2. Standard Editor - this is the standard DNN version, utilized by all versions before DNN7, with minor adjustments to make it compatible.
